# Security Policy

## Supported Versions
Security updates are currently provided only for:

- GIE AI v2.4 (Production) and above

Older versions are no longer maintained.

## Reporting a Vulnerability
If you discover a security issue, numerical instability, or a critical logical flaw
(e.g., singularity exposure, numerical collapse), please report it to:

Raymond.ma@me.com

Please avoid filing public issues for high‑severity vulnerabilities.
